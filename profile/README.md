# 0xfig Labs

[English](./README.md) | [中文](./README.zh-CN.md)

Building AI products, developer tools, and open-source infrastructure.

0xfig Labs builds practical tools for developers working with AI agents, local-first workflows, and emerging crypto infrastructure. We focus on small, composable systems that are useful from the command line, inside developer environments, and across automated workflows.

## Projects

### [tide](https://github.com/0xfig-labs/tide)

RSS for AI agents.

`tide` turns feeds into agent-friendly information streams, helping AI agents monitor updates, consume structured context, and react to changes without relying on manual browsing.

### [KeyDock](https://github.com/0xfig-labs/KeyDock)

Local-first secret workspace for AI-era developers.

`KeyDock` stores API keys locally and injects scoped environment variables into shells, scripts, and coding agents. It is designed for developers who work across many tools and need tighter control over secrets without centralizing everything in a remote service.

### [btc-connect](https://github.com/0xfig-labs/btc-connect)

Bitcoin connectivity infrastructure.

`btc-connect` is an early-stage TypeScript project exploring connection layers and developer interfaces around Bitcoin-related workflows.

## What we care about

- AI-native developer workflows
- Local-first tools and secure defaults
- Open-source infrastructure that stays simple to operate
- Interfaces that help humans and agents work together
