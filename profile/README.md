# 0xfig Labs

[English](./README.md) | [中文](./README.zh-CN.md)

Building the tool layer for AI-native builders.

0xfig Labs explores the frontier where AI agents, developer workflows, and Web3 infrastructure meet. We build open-source tools that help developers capture signals, manage secrets, automate workflows, and connect new systems without adding unnecessary complexity.

Our work is shaped by one belief: the next generation of software will be built by humans and agents working together, with tools that are local-first, programmable, secure by default, and easy to compose.

## Focus

- AI agent infrastructure and agent-friendly data flows
- Developer tools for local-first, automated workflows
- Web3 and Bitcoin connectivity primitives
- Open-source systems that are small, useful, and hackable

## Projects

### [tide](https://github.com/0xfig-labs/tide)

RSS for AI agents.

`tide` turns feeds into agent-friendly information streams. It helps agents monitor updates, consume structured context, and react to signals from the open web.

### [KeyDock](https://github.com/0xfig-labs/KeyDock)

Local-first secret workspace for AI-era developers.

`KeyDock` stores API keys locally and injects scoped environment variables into shells, scripts, and coding agents. It gives AI-era builders tighter control over secrets while keeping workflows fast and local.

### [btc-connect](https://github.com/0xfig-labs/btc-connect)

Bitcoin connectivity infrastructure.

`btc-connect` is an early-stage TypeScript project exploring developer-friendly connection layers for Bitcoin-related workflows.

## Principles

- Agent-native, human-controllable
- Local-first where it matters
- Secure defaults without heavy ceremony
- Composable tools over closed platforms
- Useful prototypes that can grow into durable infrastructure
