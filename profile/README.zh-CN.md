# 0xfig Labs

[English](./README.md) | [中文](./README.zh-CN.md)

为 AI-native builders 构建工具层。

0xfig Labs 探索 AI Agents、开发者工作流与 Web3 基础设施的交汇处。我们构建开源工具，帮助开发者捕捉信号、管理 secrets、自动化工作流，并用更低复杂度连接新的系统。

我们的核心判断是：下一代软件会由人类开发者和 Agent 协同构建，而它们需要本地优先、可编程、默认安全、易于组合的工具。

## 关注方向

- AI Agent 基础设施与 Agent-friendly 数据流
- 面向本地优先和自动化工作流的开发者工具
- Web3 与 Bitcoin 连接原语
- 小而有用、可 hack、可演进的开源系统

## 项目

### [tide](https://github.com/0xfig-labs/tide)

面向 AI Agent 的 RSS。

`tide` 将订阅源转化为 Agent 友好的信息流，帮助 Agent 监听更新、消费结构化上下文，并从开放网络的信号中触发后续动作。

### [KeyDock](https://github.com/0xfig-labs/KeyDock)

面向 AI 时代开发者的本地优先密钥工作区。

`KeyDock` 将 API keys 保存在本地，并按作用域注入到 shell、脚本和 coding agents 中。它让 AI 时代的 builders 在保持本地快速工作流的同时，更可控地管理 secrets。

### [btc-connect](https://github.com/0xfig-labs/btc-connect)

Bitcoin 连接基础设施。

`btc-connect` 是一个早期 TypeScript 项目，探索面向 Bitcoin 工作流的开发者友好连接层。

## 原则

- Agent-native，但始终由人类掌控
- 在关键位置坚持 local-first
- 默认安全，但不制造繁琐仪式
- 优先构建可组合工具，而不是封闭平台
- 从有用的原型出发，演进成可靠基础设施
