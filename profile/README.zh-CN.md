# 0xfig Labs

[English](./README.md) | [中文](./README.zh-CN.md)

构建 AI 产品、开发者工具和开源基础设施。

0xfig Labs 致力于构建面向 AI Agent、本地优先工作流和新兴加密基础设施的实用工具。我们关注小而可组合、可自动化、真正服务开发者日常工作的系统，既能在命令行里使用，也能嵌入开发环境和自动化工作流。

## 项目

### [tide](https://github.com/0xfig-labs/tide)

面向 AI Agent 的 RSS。

`tide` 将订阅源转化为 Agent 友好的信息流，帮助 AI Agent 监听更新、消费结构化上下文，并围绕信息变化执行后续任务。

### [KeyDock](https://github.com/0xfig-labs/KeyDock)

面向 AI 时代开发者的本地优先密钥工作区。

`KeyDock` 将 API keys 保存在本地，并按作用域注入到 shell、脚本和 coding agents 中。它面向同时使用多种开发工具的开发者，帮助他们在不把所有 secrets 集中托管到远程服务的前提下，更可控地管理密钥。

### [btc-connect](https://github.com/0xfig-labs/btc-connect)

Bitcoin 连接基础设施。

`btc-connect` 是一个早期 TypeScript 项目，探索围绕 Bitcoin 工作流的连接层与开发者接口。

## 我们关注什么

- 面向 AI 的开发者工作流
- 本地优先工具与安全默认值
- 简单可运维的开源基础设施
- 让人类开发者与 Agent 更好协作的接口
